# MyWebApp - 用户手册

## 1. 访问方式
- **线上地址**：[https://mywebapp.com](https://mywebapp.com)
- **本地运行**：
  ```bash
  npm install
  npm run dev
  ```
## 2. 功能介绍
- **登录/注册**：输入用户名密码即可进入系统。
- **数据查询**：在搜索框输入关键词，点击“搜索”查看结果。
- **数据导出**：支持CSV格式导出。

## 4. 常见问题
Q: 登录失败怎么办？  
A: 检查用户名密码是否正确，或联系管理员重置密码。
